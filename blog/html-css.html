Margin

The margin property defines the outermost portion of the box model, creating space around an element, outside of any defined borders. With CSS, you have full control over the margins. There are properties for setting the margin for each side of an element (top, right, bottom, and left).

.box {
margin-top: 20px;
margin-right: 10px;
margin-bottom: 20px;
margin-left: 10px;
}


Border

The CSS border properties allow you to define the style, width, and color of an element’s border. Some examples are:

Border style

border-style: solid;


border-style: dashed ;


border-style: groove;


Padding

The padding property in CSS controls the space between an element’s content and its border, forming the innermost layer of the box model. It creates internal spacing within the element, but does not affect the margins or external spacing around the element.

Padding values are defined using either length units or percentages, and they cannot be negative.

.box {
padding-top: 20px;
padding-right: 10px;
padding-bottom: 20px;
padding-left: 10px;
}
