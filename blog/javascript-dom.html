JS & DOM

An analogy to describe JavaScript and its relationship to HTML and CSS.

HTML, CSS, and JavaScript work together like building a house. HTML is like the walls, floors, doors, and overall structure, giving the house its layout. CSS is the paint, colors, and furniture, making everything look nice. JavaScript is like the electrical system and plumbing, bringing the house to life and making it functional.


Explain control flow and loops using an example process from everyday.

Doing laundry is another example to explain control flow and loops. Control flow is like following each step in order: first, gather the dirty clothes, sort them by color, wash them in the machine, add detergent, and then dry the clothes. Each step must be done in sequence. Loops is like folding clothes, you start with one clothe and then another and another till all the clothes are folded.


Describe what the DOM is and an example of how you might interact with it.

The DOM (Document Object Model) is a way for web browsers to represent the structure of a webpage. It turns the HTML into a tree of objects that you can manipulate with JavaScript.

document.getElementById("myButton").onclick = function() {
document.getElementById("myParagraph").innerText = "Hello, world!"; } ;

Here, JavaScript finds the button and paragraph by their IDs and updates the paragraph’s text to “Hello, world!” when the button is clicked.



Explain the difference between accessing data from arrays and objects.

let fruits = ["apple", "banana", "cherry"];
let firstFruit = fruits[0];
// "apple"

Arrays are like, list where you use number to find items. To get first item you use an index of 0.


let person = { name: "Alice", age: 30 };
let personName = person.name;
// "Alice"

Objects are like dictionaries with keys and values. You use the key to find the value.


Explain what functions are and why they are helpful.

A function is a piece of code that does a specific job. You can use it whenever you want, so you do not have to write again and again. Use just need to call the function when ever needed.
